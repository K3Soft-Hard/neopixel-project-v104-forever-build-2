## about this project

this project is for neopixel and display. It can be controlled via the second microbit. Some effects use a microphone.

## Edit this project

To edit this repository in MakeCode.

* open [https://makecode.microbit.org/](https://makecode.microbit.org/)
* click on **Import** then click on **Import URL**
* paste **https://github.com/k3soft-hard/neopixel-project-v104-forever-build-2** and click import
